Калькулятор на rust, с автоматическим определением последовательности действий
Реализован только для целых чисел и с правильным расставлением знаков, то есть вариант -5+8 или (5+2)4 -- не пройдет, будет вызвана паника при решении
В случае необходимости могу быстро это сделать
Пример, верного примера:
58*31+85*(10/10-3)/7*7
